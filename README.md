# Module6-7

## Структура

Заказ (класс)
- Тип доставки (абстрактный класс)
	1. На дом (наследованный класс)
	2. Точка выдачи (наследованный класс)
	3. Магазин (наследованный класс)
- Номер заказа (Guid, скрытый, назначается через свойство)
- Содержимое заказа (массив структур)
	1. Тип
	2. Модель
	3. Количество
	4. Стоимость
	5. Полная стоимость
- Краткое описание

### Базоый уровень
* **наследование**
* **абстрактные классы или элементы**
* **инкапсуляция**
* **переопределение свойств/методов**
* **4 собственных класса**
* **конструкторы с параметрами**
* **обобщения**
* **свойства**
* **композиции классов**

### Средний уровень
* статические классы или элементы
* обобщённые методы
* корректное использование абстрактных классов
* корректное применение модификаторов
* логика в геттерах и сеттерах

### Продвинутый уровень
* методы расширения
* наследование обобщений
* **агрегация классов**
* индексаторы
* перегруженные операторы
